# Zuckey and Gami's podcast page
## Development server
- `git clone git@github.com:zuckeyM-17/shiganai-radio-page.git`
- `cd shiganai-radio-page`
- `npm install`
- `ng serve`
- access to http://localhost:4200/

## Production Build
- `npm run build-prod`

## Production Build & Deploy
- Production
    - `npm run deploy-prod`
- Staging
    - `npm run deploy-staging`
